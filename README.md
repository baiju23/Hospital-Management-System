# Hospital Management System

A Python-based Hospital Management System project developed to manage patient records efficiently.

## Features
- User Login Authentication
- Add Patient Records
- Search Patient Details
- Update Patient Information
- Delete Patient Records
- File Handling
- Exception Handling
- Patient ID Generation
- Secure 3-Attempt Login System

## Technologies Used
- Python
- Lists
- Dictionaries
- Functions
- File Handling

## Future Improvements
- Database Integration
- GUI Interface
- Appointment Management
- Billing System
